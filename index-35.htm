Multi-Cloud Object
Storage

MinIO offers high-performance, S3 compatible object storage.
Native to Kubernetes, MinIO is the only object storage suite available on
every public cloud, every Kubernetes distribution, the private cloud and the
edge. MinIO is software-defined and is 100% open source under GNU AGPL v3.

True Multi-Cloud

Multi-cloud object storage allows enterprises to build AWS S3 compatible data infrastructure on any cloud.
The result is a consistent, portable interface to your data and applications - meaning you can run anywhere,
from the edge to the public cloud without changing a line of code.

Public Cloud Private Cloud Colo The Edge
Powered by Kubernetes, MinIO delivers scalable, secure, S3 compatible object storage to every public cloud.
Free yourself from vendor lock in and treat the cloud for what it is - commodity compute, networking and drives.
From OpenShift to Tanzu, MinIO is the only object store to be a foundational part of the infrastructure for
leading Kubernetes distributions. With its massive portfolio of integrated applications MinIO completes
the software-defined picture.
MinIO has powered the leased infrastructure market since its inception, delivering throughput performance
for large scale data infrastructure. With MinIO, Kubernetes and the leased infrastructure enterprise get the
benefit of public cloud infrastructure with the control of the private cloud.
At less than 100MB for the full binary, MinIO is capable of powering object storage anywhere - from
ARM SOCs, 5G POPs and edge caching devices to mini-datacenters. It is why MinIO dominates the edge storage market.

Active Active Replication
MinIO offers multi-site, strictly consistent replication for mission critical production environments.
Identity & Access Management
MinIO extends AWS IAM compatibility with support for popular external identity providers.
Encryption
MinIO’s state-of-the-art encryption include AES-256-GCM, ChaCha20-Poly1305, and AES-CBC.
Bucket & Object Immutability
MinIO supports object locking, retention, legal holds, governance, and compliance.
Bucket & Object Versioning
MinIO supports sophisticated, independently versioned objects following Amazon’s S3 structure/implementation.
Data Life Cycle Management & Tiering
MinIO offers a unique suite of features to protect data within and across clouds - both public and private.
Automated Data Management Interfaces
MinIO’s data management interfaces deliver granular, performant and scalable object storage management.
Monitoring
MinIO cluster visibility with detailed storage performance monitoring, metrics, and per-operation logging.
Scalability
MinIO scales through server pools. Each server pool is an independent set of nodes with their own compute, network and storage resources.

Performance Benchmark

MinIO pioneered high-performance object storage and remains the fastest object store in the market with GET/PUT throughput of 325 and 165 GiB/sec respectively on just 32 nodes of NVMe. These speeds enable any workload to run on MinIO - from advanced analytics to AI/ML. See all of our benchmarks in our resource library.
